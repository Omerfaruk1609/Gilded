# 🏺 Gold: Kintsugi Space

**"Yaralarınız, ışığın içeri girdiği yerdir."**

Gold (Kintsugi Space), insanların kırgınlıklarını paylaştığı, topluluğun ise bu kırgınlıkları "altın dikişlerle" (desteklerle) onardığı premium bir sosyal platformdur. Kintsugi felsefesinden (kırılan objeleri altınla onarma sanatı) ilham alan bu proje, kusurların güzelliğini ve dayanışmanın gücünü kutlar.

---

## ✨ Temel Özellikler

### 🧘‍♂️ Kintsugi Deneyimi
- **Dinamik Onarım Görselleştirmesi:** Paylaşılan her "kırık parça" (post), aldığı destek sayısı arttıkça üzerindeki altın çatlaklar birleşir ve post görsel olarak onarılır.
- **Altın Dikiş (Stitch):** Kullanıcılar birbirlerinin acılarına ortak olmak için "Altınla Dik" butonunu kullanır.

### 🛡️ Gelişmiş Moderasyon ve Güvenlik
- **Üç Kademeli Rol Sistemi:**
  - **Yönetici (Admin):** Tam platform kontrolü, kullanıcı yetkilendirme, içerik silme ve istatistik takibi.
  - **Bilge (Sage):** Topluluğun rehberleri. "Bilgelik Panosu"na özel içerik gönderme yetkisine sahipler.
  - **Üye (Member):** Standart topluluk üyeleri.
- **Küfür Filtresi:** Paylaşımlar ve yorumlar otomatik olarak denetlenir, topluluk kurallarına aykırı içerikler engellenir.
- **Tam Anonimlik:** Tüm paylaşımlar ve yorumlar varsayılan olarak anonimdir, güvenli bir paylaşım alanı sunar.

### 📊 Yönetim Merkezi (Admin Panel)
- **Canlı İstatistikler:** Toplam kullanıcı, post, onarılan içerik ve dikiş sayılarını anlık takip edin.
- **İçerik Denetimi:** Tüm postları ve yorumları tek bir merkezden yönetin veya silin.
- **Kullanıcı Yönetimi:** Üyeleri "Bilge" veya "Yönetici" olarak yetkilendirin.

### 🔔 Etkileşim ve Bildirimler
- **Gerçek Zamanlı Bildirimler:** Birisi postunuza destek verdiğinde veya yorum yaptığında anında haberdar olun.
- **Bilgelik Panosu:** Sadece yöneticiler ve bilgeler tarafından paylaşılan, ilham verici içeriklerin toplandığı özel alan.

---

## 🛠️ Teknoloji Yığını

- **Frontend:** React 18, Vite, Material UI (MUI), Framer Motion
- **Backend:** Node.js, Express.js
- **Veritabanı:** SQLite (Better-SQLite3)
- **Güvenlik:** Bcrypt.js (Şifre Hashleme), Role-Based Access Control (RBAC)
- **Styling:** Vanilla CSS & MUI System

## 🚀 Kurulum

### 1. Depoyu Klonlayın
```bash
git clone https://github.com/Omerfaruk1609/Gilded.git
cd Gilded
```

### 2. Bağımlılıkları Yükleyin
```bash
npm install
```

### 3. Sunucuyu Başlatın
```bash
# Hem backend hem frontend'i aynı anda başlatmak için:
npm run dev
```
